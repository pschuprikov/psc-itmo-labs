\chapter{Предлагаемый метод}
В данной главе будет изложен предлагаемый метод решения поставленной задачи, который в корне отличается от методов, использующих растеризацию поверхности, подверженной действию эффекта (например карты расстояний или карты направлений <<на берег>>). Главным преимуществом метода является пропорциональный числу вершин в контуре береговой линии размер памяти, требуемый для хранения вспомогательных данных.

Описание состоит из следующих разделов:
\begin{enumerate}
\item \b{Построение береговых координат.} Основа предлагаемого метода --- построение непрерывных 
текстурных координат, ориентированных вдоль береговой линии, с использованием такого метода 
представления полигона, как \emph{кусочно-линейный скелет} (англ.~Straight Skeleton). 
\cite{skeleton,skeleton_azmanov,skeleton_theory}.
\item \b{Метод наложения эффектов вдоль протяженной береговой линии.} Комбинации
техники из предыдущего пункта и метода \emph{объемных наклеек} (англ.~volume decal) \cite{volume}.
\item \b{Эффект прибоя и прибрежной пены.} Использование различных псевдослучайных шумов и их комбинации
для достижения реалистичной визуализации прибоя.
\item \b{Эффекты на берегу} Накат/скат пены, <<мокрый песок>>.
\end{enumerate}
